# Exercises - Module 01

Each folder contains an isolated exercise for a given lesson.

## Exercise Structure

```
ex01-health-controller/
├── README.md           # Task + hints
├── src/                # Your code
└── solution/           # Solution (open if you get stuck)
```

## Workflow

1. Read the lesson in `lessons/`
2. Perform the exercise
3. Check if it works
4. Compare with the solution (optional)
