# Exercises - Module 01

Każdy folder zawiera izolowane ćwiczenie do danej lekcji.

## Struktura ćwiczenia

```
ex01-health-controller/
├── README.md           # Zadanie + hints
├── src/                # Twój kod
└── solution/           # Rozwiązanie (otwórz jak utkniesz)
```

## Workflow

1. Przeczytaj lekcję w `lessons/`
2. Wykonaj ćwiczenie
3. Sprawdź czy działa
4. Porównaj z solution (opcjonalnie)
